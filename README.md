## 要件
- PHP 5.6 以上
- WordPress 3.9.3 以上

## 導入手順
1. ZIPダウンロード  
2. wp-content/plugins に展開  
3. 管理画面から有効化  

## 使用方法
1. 投稿画面のエディタでアニメーションを追加したい文章をマウスで選択
2. 「マーカーアニメーション」ボタンを押下 (黄色く色がついた部分がアニメーションの対象になります)
3. アニメーションを外したい場合は対象の文にカーソルを合わせた状態で「マーカーアニメーション」ボタンを押下
